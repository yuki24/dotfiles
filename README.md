My Emacs Lisp Installation
==========================

do the followings.

    cd ~/
    git clone git://github.com/yuki24/dotfiles.git site-lisp
    cp site-lisp/.emacs.el .
    cd .emacs.d
    git clone git://github.com/eschulte/rinari.git
    cd rinari
    git submodule init
    git submodule update
